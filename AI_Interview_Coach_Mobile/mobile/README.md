# AI Interview Coach Mobile

Aplicacion movil (Expo + React Native) para simular entrevistas tecnicas y recibir feedback automatico.

## Requisitos

- Node.js 20+
- npm 10+

## Configuracion

1. Copia el archivo de entorno:

```bash
cp .env.example .env
```

2. Instala dependencias:

```bash
npm install
```

3. Arranca la app:

```bash
npm run start
```

La app espera la API en `EXPO_PUBLIC_API_URL`.
